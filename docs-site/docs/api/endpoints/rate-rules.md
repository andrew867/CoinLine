# Rate rules API

**Route prefix:** `/api/rate-rules`

| Method | Path |
|--------|------|
| PUT | `/api/rate-rules/{id}` | Update existing rule |

Creation of rules for a version is via **`POST /api/rate-plans/{planId}/versions/{versionId}/rules`** — see [Rate plans](rate-plans.md).

**Entity:** `RateRule`
