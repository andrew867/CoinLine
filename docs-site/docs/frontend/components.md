# Components

Notable:

- **`Layout`** — shell navigation.
- **`HexViewer`** — raw bytes / hex display for diagnostics.
- **`JsonPanel`** — generic GET-and-render JSON (used for legacy shortcuts).
- **`GlobalSearch`** — operator search.
- **`SimulationBanner`**, **`FirmwareLiveFlashBanner`** — safety messaging.
- **`AppStates`** — loading / error / empty patterns.
