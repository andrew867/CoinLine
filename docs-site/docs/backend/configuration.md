# Configuration

Primary sources:

- `appsettings.json`, `appsettings.Development.json`
- Environment variables / `__` nesting for overrides
- Documented in [Environment variables](../reference/environment-variables.md)

Sections include **`Security`**, **`Platform`** (CORS, retention, query limits, seed), **`Firmware`**, **`CardPayments`**, **`Observability`**.
